# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it by sending an email to security@ingeniousclan.com. Please do NOT open a public issue or pull request for security vulnerabilities.

Include as much information as possible:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any proposed fix

## Supported Versions

| Version | Supported |
| --- | --- |
| latest | Yes |
| older versions | On request |

## Security Considerations

- Keep your dependencies updated
- Review security advisories regularly
- Follow secure coding practices

Thank you for helping keep this project secure!
